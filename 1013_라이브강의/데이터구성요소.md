# 데이터 구성요소 (Feature/Label)
- 데이터가 왜 중요한가?
    - 머신러닝은 규칙을 직접 코딩하지 않고, 데이터에서 규칙을 학습
    - 데이터(Feature, Label)의 분포와 관계가 머신러닝의 학습 결과를 결정
- Feature(피처, 특성)
    - 모델이 예측에 사용하는 입력정보
    - 예측, 판단의 근거/단서
- Label(라벨, 목표값)
    - 모델이 예측하려는 정답
    - 학습의 목표값

# ML 실생활 에시
- 예시 1 : 유튜브 추천
    - Feature : 각 영상들의 정보, 사용자 정보
    - Label : 영상에 대한 사용자 피드백
- 예시 2 : 스팸메일 분류
    - Feature : 메일 제목, 발신자, 단어 빈도
    - Label : 스팸 / 정상
